  <h1 style="margin:0 0 8px;">EDA BOM Converter for digikey and Mouser (OrCAD, Altium, KiCad, PADS)</h1>

  <p style="color:#374151;line-height:1.6;margin:0 0 12px;">
    Convert BOM files exported from <strong>OrCAD, Altium, KiCad, and PADS</strong> into distributor upload-ready formats.
    This converter is <strong>EDA-agnostic</strong>: you do <strong>not</strong> need to select which EDA tool generated the BOM.
    Upload any BOM, select the <strong>header row</strong>, optionally adjust <strong>MPN / Quantity</strong> using dropdowns, then download
    <strong>Mouser</strong> or <strong>Digi-Key</strong> BOM templates.
  </p>

  <div style="background:#f9fafb;border-radius:12px;border:1px solid #e5e7eb;margin:12px 0;padding:12px;">
    <p style="color:#111827;line-height:1.6;margin:0;">
      <strong>Status:</strong> Active (MVP). Outputs:
      <strong>Mouser BOM template (.xlsx)</strong> + <strong>Digi-Key BOM template (.xlsx)</strong>.
    </p>
    <p style="color:#6b7280;font-size:13px;line-height:1.6;margin:6px 0 0;">
      Note: This tool runs on Google Apps Script. A Google permission screen may appear on first use.
    </p>
  </div>

  <hr style="border:0;border-top:1px solid #e5e7eb;margin:18px 0;" />

  <h2 style="margin:0 0 10px;">How it works</h2>
  <ol style="color:#374151;line-height:1.7;margin:0 0 14px 18px;">
    <li>Upload your BOM file (<strong>.csv .txt .tsv .xls .xlsx</strong>).</li>
    <li>Check the preview and click the <strong>header row</strong> (clicking any cell selects the entire row).</li>
    <li>The tool auto-detects <strong>MPN</strong> and <strong>Quantity</strong> columns from the header text (you can override via dropdowns).</li>
    <li>Download either <strong>Mouser .xlsx</strong> or <strong>Digi-Key .xlsx</strong>.</li>
  </ol>

  <div style="background:#ffffff;border-radius:12px;border:1px solid #e5e7eb;margin:12px 0;padding:12px;">
    <p style="color:#111827;line-height:1.6;margin:0;">
      <strong>Digi-Key output requirement:</strong> Header on <strong>Row 2</strong> (Row 1 blank),
      Column A header must be exactly <strong>Manufacturer Product Number</strong>, Column B header must be exactly <strong>Quantity</strong>.
    </p>
  </div>
